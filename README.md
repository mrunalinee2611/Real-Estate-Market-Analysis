# 🏡 Real Estate Market Analysis Dashboard

A data-driven analysis project that explores real estate pricing trends across cities and states using SQL, Python, and Power BI. This project focuses on identifying high-value markets and investment opportunities using key metrics such as average price and price per square foot.

---

## 🚀 Project Overview

This project analyzes real-world real estate data to answer key business questions:

- Which states have the highest property prices?
- Which cities offer the best investment opportunities?
- How does price vary with property size and number of bedrooms?
- Which locations have the highest price per square foot?

The insights are visualized through an interactive Power BI dashboard.

---

## 🛠️ Tech Stack

- **Python (Pandas)** → Data loading & preprocessing  
- **SQL (SQLite via Jupyter Notebook)** → Data analysis  
- **Power BI** → Dashboard & visualization  
- **Excel** → Initial data cleaning  

---

## 📊 Key Features

- Cleaned and processed large real estate dataset
- Performed SQL-based exploratory data analysis
- Built calculated metrics such as:
  - Average Property Price
  - Average Property Size
  - Price per Square Foot
- Designed an **interactive Power BI dashboard** with:
  - KPI cards
  - State and city-level comparisons
  - Trend analysis (price vs bedrooms)
  - Top 10 filtering for focused insights
  - Dynamic slicers for user-driven exploration

---

## 📈 Dashboard Insights

- Identified top-performing states with the highest average property prices
- Highlighted cities with premium price per square foot (high investment potential)
- Observed correlation between property size, bedrooms, and price
- Enabled interactive filtering for deeper market exploration
